We can see that the distribution of the target class ('Cover_Type') is highly imbalanced. The proportions of each class are as follows:

  • Class 2: 48.8%
  • Class 1: 36.5%
  • Class 3: 6.2%
  • Class 7: 3.5%
  • Class 6: 3.0%
  • Class 5: 1.6%
  • Class 4: 0.5%

This indicates that some classes are significantly underrepresented compared to others, which could impact the performance of the model.

We observe that there are no missing values in the dataset. However, for the categorical features, 0 represents absence and 1 represents presence.

Notably, most of the soil types are absent (0) in almost every sample. Therefore, we can eliminate all soil type columns except for Soil_Type23 (present in 9.9% of samples) and Soil_Type29 (present in 19.8% of samples).
Additionally, we can remove the Wilderness_Area2 column. We will keep Wilderness_Area4 because of it's correlations with the target value.

Regarding outliers, we have identified the following thresholds for certain features:

  • Slope: Values above 40
  • Horizontal_Distance_To_Hydrology: Values above 1000
  • Vertical_Distance_To_Hydrology: Values above 300 and values below -100
  • Hillshade_9am: Values below 100
  • Hillshade_Noon: Values below 150

These thresholds help in identifying potential outliers that might need further investigation or removal.

One key takeaway from the profile report is that the target class does not have a high correlation with most of the other features. However, there are two notable exceptions:

  • Elevation: -49.6%
  • Wilderness_Area4: 72.9%

Step 5: Analysis and Discussion¶

Comparison of Logistic Regression vs k-NN

Performance Comparison:

Logistic Regression achieved an accuracy of 0.68, while k-NN also achieved an accuracy of 0.68. Both models performed similarly in terms of accuracy, but their performance metrics (precision, recall, F1-score) show some differences for individual classes.

Specific Classes and Difficulty:

Classes 3, 4, and 5 were particularly challenging for both models.

For class 3: Logistic regression achieved a precision of 0.81 but a recall of 0.03, showing it had a high false-negative rate. For class 4: Logistic regression and k-NN both struggled with very low precision and recall scores. For class 5: Both models had very low performance, with Logistic regression having 0 precision and k-NN struggling similarly.

k-NN had a similar pattern to Logistic Regression, but there were specific differences:

Class 2: k-NN had slightly better recall (0.54 vs 0.54 for logistic regression). Class 1: Logistic regression had a better recall score (0.80 vs 0.80) but similar precision.
